# Curso de Node.js



## Descripción general

## Curso 1

Contiene una colección de ejercicios prácticos de introducción a Node.js. El objetivo es explorar conceptos fundamentales del entorno, como el uso del sistema de archivos, rutas, procesos, creación de servidores HTTP y el manejo de módulos en CommonJS y ES Modules.

Los archivos del proyecto muestran diferentes funcionalidades de Node.js de forma sencilla y progresiva. A través de ejemplos concretos se abordan temas como:

- Información del sistema operativo
- Lectura y análisis de archivos
- Manejo de rutas y directorios
- Uso del objeto process
- Creación de servidores web
- Diferencias entre CommonJS y ES Modules

## Requisitos

- Node.js 16 o superior
- npm

## Cómo ejecutar los ejemplos

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Ejecutar alguno de los scripts de ejemplo:
   ```bash
   node 1.os-info.js
   node 3.fs-readFile.js
   node 8.ls-advanced.js .
   node 9.http.js
   ```

## Notas

Este proyecto sirve como base introductoria para comprender conceptos esenciales de Node.js y su uso cotidiano en aplicaciones backend.
